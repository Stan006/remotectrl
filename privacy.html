Privacy By Design

RemotePC is engineered so your data never passes through our servers. No accounts. No telemetry. No cloud relay.

1. Overview

RemotePC consists of an Android controller, a Windows listener, and this website. All communication is direct peer-to-peer over your local network.

2. What we do not collect

  • Personal information
  • Usage metrics or telemetry
  • Advertising identifiers
  • Location data

3. Android permissions

  • INTERNET — Local network connection
  • ACCESS_WIFI_STATE — Network validation
  • Storage (optional) — File transfer only

4. Website & analytics

The website uses analytics only with consent and only to measure anonymous traffic volume.
